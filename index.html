Summary: Latitude vs. Weather Factors

Project Scope:

The purpose of this project was to analyze how city weather is affected by its location to the equator. This was accomplished by acquiring data from the OpenWeatherMap API. Data for over 500 cities was pulled from this API to establish a dataset for further analysis.

To complete further analysis of this dataset, Matplotlib was utilized to plot latitude vs various weather factors. These factors included: humidity, temperature, wind speed, and cloudiness. Per this dashboard, the source data as well as the visualizations generated from the analysis process can be accessed. In addition, further insight of any observed trends and correlations is provided.
